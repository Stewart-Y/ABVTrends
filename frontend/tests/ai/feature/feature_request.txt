Users should be able to save products to a personal watchlist, receive alerts when trend score rises by +15, and view watchlist changes over time.

## Acceptance Criteria
- User can add/remove products from watchlist
- User receives notification when tracked product score increases by 15+
- User can view history of score changes for watchlist items
- Watchlist persists across sessions
- Maximum 50 items per watchlist
